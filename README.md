# JavaProject
A fast-paced brain training game built with Java Swing. This application tests your reaction speed and cognitive flexibility based on the Stroop Effect.

## üéÆ How to Play

1.  Click **"START GAME"** to begin.
2.  A word will appear on the screen (e.g., "RED").
3.  The text will be painted in a specific color (e.g., Blue ink).
4.  **Your Goal:** Determine if the **Text Meaning** matches the **Ink Color**.
    * Click **MATCH** if they are the same.
    * Click **NO MATCH** if they are different.
5.  You have **30 seconds** to get the highest score possible!
